- 07/23確定終値: 158円（本メモ1-1章、2回連続一致確認済み）

### 3-2. Python再計算

```python
shares = 54_583_175
equity_q1 = 609_000_000       # 自己資本（26.01-03期、2026-05-15開示）
bps = equity_q1 / shares       # = 11.1573...円

price_0718 = 184
price_now  = 158

mktcap_0718 = shares * price_0718   # = 10,043,304,200円 => 100.43億円
mktcap_now  = shares * price_now    # =  8,624,141,650円 =>  86.24億円

pbr_0718 = price_0718 / bps          # = 16.49倍
pbr_now  = price_now  / bps          # = 14.16倍

pct_change_price  = (price_now - price_0718) / price_0718 * 100   # = -14.13%
pct_change_mktcap = (mktcap_now - mktcap_0718) / mktcap_0718 * 100  # = -14.13%
```

**実行結果**:

| 項目 | 2026-07-18メモ時点（184円） | 2026-07-23本日時点（158円） |
|---|---|---|
| 時価総額 | 100.43億円 | **86.24億円**（kabutan本日表示値86.2億円と一致・検算OK） |
| PBR | 16.49倍 | **14.16倍**（kabutan本日表示値14.16倍と一致・検算OK） |
| 株価騰落率 | - | **-14.13%**（184円→158円） |

**参考（別基準・注記）**: kabutan.jp財務推移表が独自に掲載する「1株純資産」欄の直近値は12.65円
（26.01-03期、上記自己資本609百万円を発行済株式数の別時点値で除した社内集計値とみられ、上記の
