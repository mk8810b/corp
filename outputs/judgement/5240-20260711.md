## 1. 判定

**PASS**（見送り。BUY・WATCHいずれも採用しない）

「欠点が見当たらないからPASS」という消極的な判定ではなく、以下の却下理由が積み重なっていることを
理由とする（D-012はBUYの消極的採用のみを禁止しており、PASSの理由の質を問わない）:

1. 2026-07-10のストップ高・出来高急増（前日終値130円→終値180円、+38.46%〔Python再計算済み、
   下記4節参照〕）を説明する企業固有の適時開示・ニュース・材料が**取得不能**（EDINET直近14/60/120日間
   に該当書類なし、kabutan・Yahoo!ファイナンスのニュース欄にも個別材料なし。企業調査メモ4節）。
   D-012の4つの積極的採用理由のうち「カタリスト特定済み」は明確に不成立。
2. 3期連続（第11〜13期、2023〜2025年12月期）で営業損失・経常損失・純損失・営業CFマイナスを計上
   （企業調査メモ5節）。有価証券報告書「事業等のリスク (6) 重要事象等について」に
   「継続企業の前提に重要な疑義を生じさせる事象又は状況が存在しております」との記載がある
   （企業調査メモ7-1節）。正式な財務諸表注記としての「継続企業の前提に関する注記」は付されておらず
   監査意見も無限定適正だが、D-015テール脆弱性フラグ(a)「継続企業の前提に関する**注記または
   重要事象の記載**」には該当する（フラグ文言は注記と重要事象記載のいずれかで発動するため）。
   仮にBUYを検討する場合でも確信度「高」（10%枠）は提案できない構造的制約がある。
3. 直近開示の2026年12月期第1四半期（2026年1-3月期）実績は、売上高133百万円で前年同期比▲68.0%
   （Python再計算済み、下記4節参照）、営業損益・経常損益・最終損益いずれも「赤字拡大」で着地
   （企業調査メモ5-2節）。会社側の通期予想は売上高858百万円（前期比▲12.4%、赤字縮小方向）
   であり、直近四半期の実績トレンドは通期会社予想の方向性と逆行している。D-012の積極的採用理由
   「業績進捗が会社予想を上回る」は不成立であるばかりか、むしろ会社予想に対する下振れ懸念が
   ある状況。
4. バリュエーション面では、PERは赤字のため算定不可、PBR2.05倍（企業調査メモ3節）のみが算定できる
   指標であり、比較対象3社（シャノン・メタリアル・CRIミドルウェア）についてはPER/PBRの具体的数値
   まで深掘りされておらず（企業調査メモ8節は時価総額比較に留まる）、「同業比バリュエーション
   下位」を具体的な数値根拠をもって明記することができない。
5. 直近の適時開示は約2ヶ月前（2026-05-15の第1四半期決算短信）であり、本日時点で企業調査部が
   当日中に新たに読み込んで発見した未認知の事実（エッジ仮説D-012の核心）も存在しない。
   4つの積極的採用理由のいずれも企業調査メモの事実で具体的に満たせないため、BUYの提案は
   D-012違反（消極的採用）になる。
6. 上記2・3の財務トレンド（3期連続赤字、継続企業の前提に関する重要事象の記載、直近四半期の
   赤字拡大）は、カタリスト不明のまま単発的に株価が急騰した銘柄としては相対的にリスクが高い
   部類と判断する。8918（2026-07-09判断）はカタリスト不明という点は共通するが、財務健全性
   （自己資本比率88.7%）に加えて会社側の増収増益予想という前向きな材料が残っていたためWATCH
   （監視継続）とした。本銘柄はテール脆弱性フラグ該当かつ直近四半期の実績が会社予想と逆方向に
   動いている点で8918より材料の質が劣後するため、監視継続のコストに見合う上振れ根拠が現時点で
   組み立てられないと判断し、WATCHではなくPASSとする。

## 4. 数値の再計算（Python、第2条）

企業調査メモに記載された比率のうち、本判断根拠で引用した数値をPythonで再計算し、転記ミスがない
ことを確認した（起案時に加え、校閲でも独立に再実行して一致を確認済み）。

```python
# ストップ高の変化率
prev_close = 130
stop_high = 180
pct_change = (stop_high - prev_close) / prev_close * 100  # -> 38.46...

# PTSの東証終値比
pts = 213
pct_pts = (pts - stop_high) / stop_high * 100  # -> 18.33...

# 2026年1-3月期(1Q)売上高の前年同期比
q1_2026 = 133   # 百万円
q1_2025 = 415   # 百万円（前年同期実績）
yoy = (q1_2026 - q1_2025) / q1_2025 * 100  # -> -67.95（≒-68.0）

# 通期会社予想売上高の前期比（第13期実績比）
fy_forecast = 858000     # 千円
prior_actual = 980881    # 千円（第13期実績）
fy_pct = (fy_forecast - prior_actual) / prior_actual * 100  # -> -12.53（≒-12.5）
```

実行結果: ストップ高+38.46%、PTS+18.33%、Q1売上高前年同期比▲67.95%（企業調査メモの丸め表記
▲68.0%と整合）はいずれも企業調査メモの記載値と一致。通期予想の前期比は▲12.53%（≒▲12.5%）。
企業調査メモの記載▲12.4%は、分母に980百万（980.881千円の丸め値）を用いた場合の
(858-980)/980＝▲12.45%由来の丸め方法の違いによるものであり、実質的な転記ミスではない
（校閲でも同計算で確認済み）。投入額の円換算は本稿では発生しない（PASSのため新規投入を提案
しないため、現在資金残高〔`corp/cash.md` 2026-07-09申告、3,000,000円デフォルト〕を分母とする
計算は不要）。

## 7. 判断根拠（企業調査メモの事実との対応、まとめ）

1. **カタリスト不在**: 出来高急増・ストップ高の要因が取得不能（企業調査メモ4節、EDINET
   14/60/120日間該当書類なし・個別ニュースなし）。
2. **継続企業の前提に関する重要事象**: 3期連続赤字・営業CFマイナス、有価証券報告書に重要事象の
   記載あり（企業調査メモ5節・7-1節）。D-015テール脆弱性フラグ(a)該当。
3. **直近四半期の業績悪化**: 2026年1-3月期は売上高前年同期比▲68.0%、各段階損益とも赤字拡大で
   通期会社予想（赤字縮小）と逆行（企業調査メモ5-2節、Python再計算済み・4節）。
4. **バリュエーション比較の材料不足**: PER算定不可・PBR2.05倍のみで、同業3社との具体的な
   PER/PBR比較データがなく「同業比バリュエーション下位」を具体的事実で明記できない
   （企業調査メモ3節・8節）。
5. **エッジ仮説との不整合**: 直近開示は約2ヶ月前であり、当日読み込みによる未認知の事実発見が
   ない（企業調査メモ4-2節）。

D-012が定める4つの積極的BUY採用理由（カタリスト特定済み／同業比バリュエーション下位／
業績進捗が会社予想を上回る／エッジ仮説との整合）のいずれも、企業調査メモの事実に基づいて
具体的に満たすことができない。加えてD-015テール脆弱性フラグ(a)該当・直近四半期の業績悪化という
5240固有の追加リスクが積み重なっているため、precedent（8918、WATCH）とは判定を分け、PASSとする。
