## 7. 判断根拠（D-012の4基準への当てはめ）

企業調査メモ（`outputs/research/6619-20260718.md`、as_of 2026-07-18、バッジRSH-S07）の
事実に基づき、D-012が定める4つの積極的採用理由それぞれについて、現時点で立証できるかを
検討した。

1. **カタリスト特定済み（かつ未織り込み）**: **立証できない。** 企業調査メモには、業績・
   株価を動かす具体的な予定イベント（決算発表日・提携発表等）で市場が未織り込みと
   判断できる根拠は記載されていない。イオン交換膜事業の量産開始（2024年〜）は既に開示済み
   の既知情報であり、新規性のあるカタリストとは言えない。

2. **同業比バリュエーション下位、かつ割安さを正当化する構造的欠陥が見当たらない**:
   **前段は真だが後段が明確に偽であり、基準全体を満たさない。** PBR0.24倍は同業3社
   （旭化成1.18倍・東レ0.93倍・高度紙工業2.80倍、企業調査メモ9節）の中で突出して低く、
   バリュエーション下位という事実自体は成立する。しかし企業調査メモ6-7節は「単に市場評価が
   追いついていないと単純に片付けられる状況ではなく、実在するリスク要因を伴う低PBRである」
   と明記しており、（i）売上88.3%減・純損失125億円の業績急落、（ii）継続企業の前提に関する
   重要事象等、（iii）MSワラント3本連続発行という希薄化依存、（iv）プライム市場上場維持
   基準抵触という4つの実在する構造的要因を割安さの背景として特定している。judgement.mdが
   要求する「割安さを正当化する構造的な欠陥（財務悪化・成長性喪失等）が見当たらない」との
   要件に明確に反するため、この基準は満たさないと判断する。

3. **業績進捗が会社予想を上回る**: **立証できない。むしろ逆方向の材料がある。**
   Python再計算（下記）の通り、第22期第1四半期（2026年2-4月）実績は、通期会社予想
   （売上高6,000百万円・経常損失4,400百万円、企業調査メモ4節）に対して、売上高進捗率
   23.1%（単純4等分基準25.0%とほぼ同水準）、経常損失進捗率46.1%（同25.0%を大きく超過、
   =計画より速いペースで損失を計上）となっている。売上の伸び自体（前年同期比+83.3%）は
   evidenceとして存在するが、損益面では会社予想を上回る進捗とは言えず、この基準は
   立証できない。

   ```
   q1_sales = 1386        # 百万円（第22期Q1実績、企業調査メモ4節）
   fy_sales_forecast = 6000   # 百万円（第22期会社予想）
   q1_sales_pace = q1_sales / fy_sales_forecast * 100  # = 23.1%

   q1_ord_loss = 2028         # 百万円（第22期Q1実績・経常損失）
   fy_ord_loss_forecast = 4400  # 百万円（第22期会社予想・経常損失）
   q1_loss_pace = q1_ord_loss / fy_ord_loss_forecast * 100  # = 46.1%
   ```

4. **エッジ仮説との整合（カバレッジの薄い銘柄の適時開示・決算短信を当日中に読み込み、
   広く認知されていない具体的事実を発見）**: **立証できない。** 企業調査メモが参照した
   開示は有価証券報告書（提出日2026-04-17、docID S100XZF3）が中心であり、判定基準日
   （2026-07-18）から約3ヶ月前の既存開示である。今回のスクリーニングは割安度（軸D・低PBR）
   経由の定例deep-diveであり、D-012が想定する「当日中の決算・適時開示イベント読み込み」
   という状況とは異なる。継続企業の前提に関する記載・MSワラント履歴等は本メモが企業調査
   メモを通じて把握した重要な事実だが、これらは有価証券報告書という公開性の高い定期開示に
   記載された情報であり、「広く認知されていない」との根拠を示せない。

**結論**: 4基準のいずれも企業調査メモの事実で具体的に立証できず、うち基準2（同業比
バリュエーション下位）は要件（構造的欠陥が見当たらないこと）に明確に反することが確認
できた。「却下理由が一つも見つからなければBUY」という消極的判定を待つまでもなく、本件は
却下理由（複数の実在する構造的リスク）が積極的に確認されたケースであり、PASSとする。
