### 2-1. 追加の定量的観察（新規指摘）: 価格の「全戻り」

Python再計算（下記）により、07/17発表前の水準（07/16終値157円）と本日終値（158円）がほぼ一致
していることを確認した。

```python
price_pre = 157            # 07/16終値（材料発表前）
price_0717_close = 184     # 07/17終値
price_0717_high  = 205     # 07/17高値
price_now = 158             # 07/23確定終値（2回連続一致確認値、研究メモ1章）

roundtrip_pct = (price_now - price_pre) / price_pre * 100
# => +0.64%
drawdown_from_high_pct = (price_0717_high - price_now) / price_0717_high * 100
# => 22.93%
```

**実行結果**: 07/16終値157円 → 07/23終値158円（**+0.64%**）。07/17高値205円からは**-22.93%**。
つまり6営業日を経て、株価は材料発表前とほぼ同一の水準まで戻っている。これは「押し目形成」という
中立的な字面以上に、**07/17の急騰による超過リターンが実質ゼロに帰したこと**を意味する。さらに
07/16終値157円時点のPBRを試算すると**14.07倍**（Python再計算、下記）であり、現在のPBR14.16倍と
ほぼ同水準——**すなわちこの銘柄の高PBRは今回の材料とは無関係に元々存在した構造的な割高水準**
であることが確認できる。この事実は、07/17の材料そのものが持続的な再評価を市場にもたらしていない
（一過性の投機的反応に過ぎなかった）ことを補強する新しい根拠であり、5章の上振れ余地スコア見直しの
主因とする。

```python
shares = 54_583_175
equity_q1 = 609_000_000
bps = equity_q1 / shares          # 11.1573円
pbr_pre = 157 / bps                # => 14.07倍
pbr_now = 158 / bps                # => 14.16倍
```

## 5. WATCH継続 vs PASS格下げの検討（本メモの中心論点）

BUY格上げの根拠がないことは2章で確定した。以下、WATCH継続とPASS格下げの両方を具体的に比較検討する。

- テール脆弱性フラグ(a)継続企業の前提の重要な不確実性は解消の兆しがなく、加えて自己資本比率の
  3期連続急低下・大規模な潜在希薄化（10.7%相当）が継続している。ファンダメンタルズは弱いままである。

**WATCH継続を支持する要素（本メモが採用する結論）**:
- **次回決算（2026-08-07、5216-T3）がわずか15日後に迫っている**。この決算では
  (i) これまで「未定・非開示」だった通期業績予想が初開示される可能性があり、開示されれば
  D-012基準3（業績進捗）の再照合が初めて可能になる、(ii) 継続企業の前提に関する重要な不確実性
  （フラグ(a)）の記載が更新・解消されるかどうかが確認できる、(iii) 東北電力との共同評価の進捗に
  関する定性的言及の有無が確認できる——という、投資判断上重要度の高い複数の観測点を同時に含む。
  この決算を待たずに監視対象から外すことは、D-012エッジ仮説（適時開示を当日中に読み込む）の
  運用機会を自ら手放すことになり、リスク規律上も時期尚早である。
- 5216-T2（上抜け・出来高継続シナリオ、基準205円・210円以上かつ出来高1.5倍）は引き続きOPENで
  あり、機械的な日次巡回により監視コストは既に発生している（新規の追加コストは限定的）。
- PASS格下げは「今後の材料・決算内容次第では再評価しない」という強い意思決定であり、8/7決算という
  具体的な近接イベントを目前にして行うべき判断ではない。決算後に改めてPASS/WATCH/BUYを再検討する
  方が、情報の逐次性に即した規律である。

**結論**: 2-1章の全戻り・続報ゼロという事実は、5章での上振れ余地スコアの引き下げ（次章）には
反映するが、WATCH自体の解除（PASS格下げ）は8/7決算を通過するまで見送る。8/7決算メモ更新時に、
(i)〜(iii)がいずれも不発だった場合は、その時点で改めてPASSへの格下げを積極的に検討する
（先送りではなく、次回判断メモ更新時の明示的な検討事項として本メモに記録する）。
